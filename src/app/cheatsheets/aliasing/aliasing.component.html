Aliasing

So you want to write aliases for your commonly used commands - cool! This cheatsheet details some of the nitty-gritty syntactical shenanigans that you can use to make your aliases very powerful.

When placed inline in an alias, any syntax in the syntax table will have the listed effect. For a list of built-in cvars, see the cvar table.

For a list of user-created aliases, plus help aliasing, join the Avrae Discord!

Syntax Table

Syntax Description Example
{CVAR/roll} Evaluates the cvar/rolls the input. Input: !test Rolling 1d20: {1d20}
Output: Rolling 1d20: 7
<CVAR> Prints the value of the cvar. Input: !test My strength modifier is: <strengthMod>
Output: My strength modifier is: 2
%1%, %2%, etc. Replaced with the value of the nth argument passed to the alias. Input: !alias asdf echo %2% %1%
Output (when called as !asdf first "second arg"): "second arg" first
%*% Replaced with the string following the alias. Input: !alias asdf echo %*% first
Output (when called as !asdf second "third word"): second "third word" first
&1&, &2&, etc. Replaced with the raw value of the nth argument passed to the alias. Input: !alias asdf echo &1& was the first arg
Output (when called as !asdf "hello world"): hello world was the first arg
&*& Replaced with the escaped value of the string following the alias. Input: !alias asdf echo &*& words
Output (when called as !asdf second "third word"): second \"third word\"
&ARGS& Replaced with a list of all arguments. Input: !alias asdf echo &ARGS&
Output (when called as !asdf first "second arg"): ['first', 'second arg']
{{code}} Runs the statement as raw Python-like code. See the functions table for a list of allowed functions. Input: echo {{set("strength_plus_one", strength+1)}}1 more than my strength score is {{strength_plus_one}}!
Output (given a STR of 14): 1 more than my strength score is 15!

Input: !test My roll was {{"greater than" if roll("1d20") > 10 else "less than"}} 10!
Output (assuming the roll was 8): My roll was less than 10!
